# 🏔️ ZENITH AI CLI
**La Cúspide de la Productividad en Terminal e Ingeniería Autónoma**

ZENITH es un agente de IA autónomo de alto rendimiento diseñado para vivir en tu terminal. Actúa como un **Staff Engineer**, proporcionando arquitectura profunda, refactorización segura de código y análisis de proyectos con una filosofía local-first.

---

## ✨ Pilares Centrales

### 🧠 Memoria de Grafo "Nexus"
A diferencia de los bots de chat estándar, Zenith construye un **Grafo de Conocimiento** de tu proyecto. Identifica entidades (archivos, clases, módulos) y sus relaciones, permitiéndole entender el "Por qué" de tu código, no solo el "Qué".

### 🛡️ Auto-Refactorización Segura
Deja que Zenith haga el trabajo pesado. Usando un motor de sanitización de rutas, Zenith propone cambios, te muestra un **Diff Unificado** y espera tu confirmación antes de tocar una sola línea de código.

### 🔌 Inteligencia Híbrida
* **Local (Ollama):** 100% privado, sin latencia, funciona offline.
* **Nube (Gemini 1.5):** Contexto ultra-alto para análisis de proyectos masivos.

---

## 🚀 Inicio Rápido

### 1. Instalación
**Windows (PowerShell):**
```powershell
./install.ps1
```

**macOS / Linux:**
```bash
chmod +x install.sh && ./install.sh
```

### 2. Encender el Nexus
Escanea tu proyecto para que Zenith aprenda tu arquitectura:
```bash
zenith ignite
```
